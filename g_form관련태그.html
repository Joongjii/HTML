Form 관련 태그

 Form태그는 HTML문서 내에서 사용자가 데이터를 입력할 수 있는 양식을 제공하는 태그
          Form 태그는 자식 input 태그들을 사용해 사용자가 입력한 저보를 HTTP Message로 변환하여
          서버로 전송하는 기능을 제공

            form tag의 속성
            * action 속성 : input tag에 입력된 값들을 전송할 url을 작성
            * Method 속성 : Http Method, get, post만 지원
            * enctype 속성 : 요청 url로 전송될 값들의 Mime Type을 지정
        
            Mine Type : Multipurpose Internet Mail Extensions
                메일을 통해 다양한 종류의 데이터 (이미지,영상, 음성, 파일)을 보내기 위해 사용하는 약속
                메일에 포함된 데이터가 어떠한 양식의 데이터인지를 명시하는 약속

                *HTTP통신에서 전달되는 다양한 형태의 데이터를 포함하는 약속
                        text/html : 문자데이터이고 html 양식
                        teext/css : 문자데이터리고 CSS 양식
                        audio/mpeg : 소리데이터이고 mpeg 양식

            input tag의 name 속성의 값이 요청 파라미터 이름
            input tag의 value 속성의 값이 요청 파라미터 값
